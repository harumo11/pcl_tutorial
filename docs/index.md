# Note about ROS and PCL

ここでは，ROSを勉強した時に得た知識とノウハウを
書き込んで記録しておく文章です．

特に，PCL(Point Cloud Library)は初心者向けの情報が少ないため
自身で固くしていく必要があります．その過程で得た知識を
ここに随時記録していきます．

設定の移り変わりが激しく日本語情報が全く当てにならないため
自分で，英語情報を読んで，ここに記録していきたいと思います．
